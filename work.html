A level project


In my A level project I decided to create a randomly generated maze, using a 2D array with each entry representing either a wall, door, or free space. This allowed me to apply a path-finding algorithm to it. I chose to use A-star as unlike Dijkstra's it can use heuristics to run much quicker, and allows for some variance in how the enemy behaves, as it may not always take the exact best route to you but still gets to you quickly. The program is written in python and uses the pygame library.
